Angelreise Manager – Reise löschen

Neu:
- Reiseleiter können eine komplette Reise löschen.
- Teilnehmer können keine Reisen löschen.
- Doppelte Sicherheitsabfrage.
- Bestätigung durch Eingabe von LÖSCHEN.
- Vor dem Löschen wird automatisch ein JSON-Backup heruntergeladen.
- Lokale Reise wird gelöscht.
- Cloud-Datensatz wird, soweit möglich, gelöscht.
- Danach wird automatisch eine neue leere Angelreise angelegt.
