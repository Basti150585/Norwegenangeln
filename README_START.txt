Angelreise Manager – Teilnehmer eigene Finanzen

Korrektur:
- Teilnehmer können Kosten, Anzahlungen und Abrechnung wieder auswählen.
- Teilnehmer sehen dort ausschließlich eigene Daten.
- Zahlungsstände anderer Teilnehmer bleiben verborgen.
- Reiseleiter sieht weiterhin alles.
- Teilnehmer können keine Kosten oder Anzahlungen erfassen/löschen.
